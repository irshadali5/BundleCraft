Getting Started

BundleCraft helps you convert between single-file and multi-file web projects. Whether you need to distribute a standalone HTML file or break one down for development, we've got you covered.

Bundling (Multi → Single)

Workflow
1. Upload your HTML file (entry point)
2. Upload linked CSS and JS files
3. Configure options (minify, defer, etc.)
4. Click "Bundle Files"
5. Download your single self-contained HTML

The bundler automatically detects <link> and <script> tags and replaces external references with inline content. It handles relative paths and preserves the execution order of your scripts.

Unbundling (Single → Multi)

Workflow
1. Upload your bundled HTML file
2. Choose extraction options
3. Click "Extract Files"
4. Download individual HTML, CSS, and JS files

The extractor identifies all <style> and <script> blocks, separates them into dedicated files, and updates the HTML to reference them externally. It preserves inline event handlers and critical CSS.
